## About

Discover, subscribe, and listen to podcasts.

## Features

- Podcast discovery
- Subscriptions
- Episode downloads
- Playback controls

## Installation

### From App Store
Open the **App Store** in zOS and search for "Podcasts".

### Via npm
```bash
npm install @zos-apps/podcasts
```

## Usage

- Search for podcasts
- Subscribe to follow
- Download for offline



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
